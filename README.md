# Python Even and Odd Program

## Description

This program reads numbers from a file and separates them into even and odd numbers.

## Files

* numbers.txt → input file
* even.txt → even numbers
* odd.txt → odd numbers
* separate_even_odd_numbers.py → main program

## How to Run

1. Open in PyCharm
2. Run the Python file
3. Check the output files
